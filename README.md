# StarLive
- 네이버의 vlive를 벤치마킹하여 StarLive 플랫폼을 개발 했습니다.

## 사용 기술
- SpringBoot, Gradle, Jpa, Mysql ,Java8, restApi, Mockito, Junit5
- 
## 프로젝트 목표 
- SpringBoot를 이용하여 내가 좋아하는 서비스를 MVC 패턴에 따라 만들어보면서 어떻게 동작하는지 배우는 경험 쌓기 
- Mock을 이용하여 TDD (테스트 주도 개발) 경험하기
